PSEUDO CLASSES

A continuación se muestra una lista de las pseudo classes que se pueden usar, así como ejemplos de cómo es que se usan o se ven:

  • Pseudo clases para que un link cambie de color dependiendo de la acción
    • :link - Indica que un link no ha sido visitado todavía. Instagram
    • :visited - Indica que un link ya ha sido visitado. Facebook
    • :hover - Indica que el puntero está sobre un link. Instagram
    • :active - Indica que un link ha sido seleccionado. Instagram
